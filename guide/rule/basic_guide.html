3. 작성 규칙

이미지

확장자

  • 기본 : .png
  • 사진(압축 Quality 80%) : .jpg

지원 브라우저 및 OS

  • 소문자만 사용 가능
  • 언더스코어(_) 표기법(단어와 단어 사이를 _로 구분) : bg_xx, btn_xx, ico_xx, img_xx
  • 언더스코어(_)는 최대 3단계까지만 사용 : bg_xx_xxx_x
    백그라운드 : bg_ 버튼 : btn_ 아이콘 : ico_ 이미지 : img_ GNB : gnb_
CSS

기본

  • 속성과 속성값 사이는 띄어쓰기 없고 (;) 뒤에는 1칸 띄어쓰기- {속성명:속성값; 속성명:속성값;}
  • 마지막 속성도 반드시 (;)로 끝내기
  • 다중 선택자 선언시 (,) 후 줄바꿈 하기
    .imgVisual, .stepArea {속성명:속성값; 속성명:속성값;}
  • 폰트명 이외 선택자([type=radio]), 이미지 url()등에 따옴표('', "")를 사용하지 않는다.
  • 축약속성(border:;, background:;)은 속성중 일부분 재정의시 !important 남발의 사유가 되므로 사용시 신중 할것

네이밍규칙

  • 카멜 표기법(단어와 단어 사이를 대문자로 구분) : imgVisual, stepArea
  • class 공통선언시 예외적으로 언더스코어 표기법 사용 가능 : btn_xx, btnIco_xx, btnTxt_xx
  • 숫자로 class명을 시작할 수 없음

속성선언 순서

display:; visibility:; float:; position:; top:; bottom:; left:; right:; width:; height:; margin:; padding:; border:; background:#color url() 0 0 repeat; color:; font-family:; font-size:; font-weight:; font-style:; letter-spacing:; line-height:; text-align:; vertical-align:; text-indent:; overflow:; z-index:;

CSS 속성선언 순서
순서 의미 대표되는 속성(그룹) 비고
1 표시 display / visibility
2 흐름 float 가상선택자 :after를 이용하여 해제
가상선택자 사용이 불가능 할 경우 reset.css에 정의되어 있는 clfix를 이용하여 float 해제
3 위치 position / top / bottom / left / right
4 크기 width / min-width / max-width / height / min-height / max-height
5 여백 margin / padding
6 테두리 border(그룹) 아래 순서로 축약속성 사용(축약속성 사용은 신중 - !important 남발 사유)
border-width, border-style, border-color
7 배경 background(그룹) 아래 순서로 축약속성 사용(축약속성 사용은 신중 - !important 남발 사유)
background-color, background-image, background-repeat, background-position
8 텍스트 콘텐츠 관련 color / font / letter-spacing / line-height 폰트는 축약속성을 사용하지 않으며, 다음의 순서로 정의
color, font-family, font-size, font-weight, font-style
9 정렬 text-align / vertical-align / text-indent
10 영역 box-sizing
11 넘침 overflow
12 깊이 z-index
13 기타속성

4. 기타

policy: Cross origin requests are only supported for protocol schemes 문제로 로컬에서 정상작동 안될때

  • 바탕화면에 Chrome 바로가기 생성 후
  • 마우스 우클릭 후 속성을 실행.
  • 대상 입력 필드에 --allow-file-access-from-files 추가
    "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --allow-file-access-from-files
  • 실행되고 있는 크롬 모두 종료 후 실행

크롬 실행 시 Devtools 항상 띄우기

  • 바탕화면에 Chrome 바로가기 생성 후

  • 대상 입력 필드에 --auto-open-devtools-for-tabs 추가
    "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --auto-open-devtools-for-tabs
  • 실행되고 있는 크롬 모두 종료 후 실행
